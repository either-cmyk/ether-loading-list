# 변경 이력

## v1.1.0 - 2026-05-18

### 트레이 다종 지원 + 헤더 기반 인덱싱 + PDF 행 높이 자동 조정

- **N-G 트레이 지원**: `S-QED-N-G-TP-XXX` (가습기류) prefix 추가 인식
- **혼합 트레이 지원**: 한 트레이에 G/N-G/B 다른 prefix 박스가 섞여 있어도 정상 처리
- **헤더 이름 기반 컬럼 인덱싱**: `이미지` 추가 컬럼 변형 자동 대응 (선글라스 케이스 등)
- **PDF 행 높이 자동 조정**:
  - ≤12행: 9mm / 13-16행: 8.5mm / 17-20행: 7.5mm / 20행 초과: 7mm
  - 18 SKU 이상 발주 (선글라스 다색상)도 A4 1페이지 fit
- **트레이 매핑 우선순위 명시**: G → N-G → B, 각 그룹 내 트레이번호 오름차순
- **액세서리 SKU 자동 제외**: 매핑 안 된 SKU (예: 케이스 `CN297120T8450604`)는 결과에서 자동 제외
- **검증된 신규 케이스**:
  - 131385250 (1파레트 11박스) — 선글라스 + 우양산 18 SKU, "이미지" 컬럼 변형
  - 131638295 (9파레트 140박스) — G/N-G/B 트레이 혼합, 8개 혼적박스
