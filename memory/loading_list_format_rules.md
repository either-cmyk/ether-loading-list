---
쿠팡이 2026년 4월부터 팔레트 적재리스트를 신양식으로 변경. 각 팔레트별 부착 필수.

**필드 매핑 (거래명세서 → 신양식 적재리스트):**
- 업체명 → 1) 업체정보 업체명
- 업체번호 → 1) 업체정보 업체코드
- 발주번호 → 2) 입고예약정보 요청 ID
- 납품 센터 → 2) 물류센터 (예: `인천28(INC28)`)
- 물류센터 도착예정일 → 2) 물류센터 도착예정일 (`YYYY-MM-DD`)
- 팔레트수량 → 2) 팔레트 번호 ("총 팔레트수-해당 팔레트번호" 형식, 예: `13-1`, `13-2`)
- (팔레트별 박스 수) → 2) 총 박스 (해당 파레트의 박스 수, 합계 아님)
- **상품번호 → 3) SKU ID** (8자리 숫자, 예: `43800624`)
- 상품명/옵션 → 3) 물류입고용 상품명/옵션명 (끝의 "-" 제외)
- (적재 박스번호) → 3) 박스 번호 — **쿠팡 공식 규칙**: 박스에 여러 판매 단위 상품을 적재한 경우(혼적박스)에만 `(전체박스수-하위번호)` 작성 (예: `16-15`). 단일 SKU 박스는 공란. ※ 각 혼적박스 외부에도 동일한 박스 번호 라벨 표시 필수
- 발주수량 → 3) 상품 수량
- 소비기한관리 N → 3) 소비기한/제조일자 ("-")

**전화 확인 결과 (2026-04 콜):**
"파레트별로 작성해주시는게 맞아요"
→ 박스번호 = 해당 파레트 박스수 기준 (전체 발주 합계 X)

**중요:**
- 바코드는 SKU ID 자리에 넣으면 안 됨 (옛날 잘못된 매핑). SKU ID = 상품번호
- 팔레트 한 페이지에 끝나야 함 (A4 한 장 fit-to-page)
- 사용자는 보통 단일 시트 선호. 팔레트별 시트 분리는 사용자 명시 요청 시에만
